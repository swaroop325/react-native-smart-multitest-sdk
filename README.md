# react-native-smart-multitest-sdk

Connect to Smart Multi Test Device 

## Installation

```sh
npm install react-native-smart-multitest-sdk
```

## Usage

```js
import { multiply } from "react-native-smart-multitest-sdk";

// ...

const result = await multiply(3, 7);
```
